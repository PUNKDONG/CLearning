# 题目 1：UDP 口令回显服务器 + 客户端

目标：掌握 `socket`、`bind`、`sendto`、`recvfrom`，理解无连接通信模型。

你需要写两个程序：
- `udp_server.c`
- `udp_client.c`

## 功能要求

服务端：
- 监听 `127.0.0.1:9001`
- 持续接收客户端发来的消息
- 收到消息后按下面规则回复：
  - 以 `PING ` 开头：返回 `PONG ` + 后面的内容
  - 消息等于 `TIME`：返回当前服务器时间字符串
  - 其他内容：返回 `ERR`
- 每次收到请求时，打印客户端 IP、端口、原始消息
- 服务端不能因为处理完一个客户端就退出

客户端：
- 从终端读取一行输入
- 将这一行发送给服务端
- 接收服务端回复并打印
- 打印后退出

## 约束

- 单次消息最大长度 `1024` 字节
- 要检查所有系统调用返回值
- 要处理空输入和超长输入
- 不能使用 TCP 相关接口，如 `listen`、`accept`

## 你应该重点掌握

- `struct sockaddr_in` 的填写方式
- `htons`、`htonl`、`inet_pton`
- `sendto` / `recvfrom` 的参数意义
- UDP 是“收一包、发一包”的模型

## 自测清单

- 输入 `PING hello`
- 输入 `TIME`
- 输入 `abc`
- 连续运行多个客户端，确认服务端持续工作

## 验收标准

- 服务端正常监听并回复
- 客户端能正确显示返回结果
- 异常输入不会导致崩溃
