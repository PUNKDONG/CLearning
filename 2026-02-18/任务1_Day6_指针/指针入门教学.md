# Day6 指针入门教学（重点）

## 1. 指针到底是什么

一句话：**指针是“存地址的变量”**。

- 普通变量存“值”，比如 `int a = 10;`
- 指针变量存“地址”，比如 `int *p = &a;`

这里：
- `a` 的值是 `10`
- `&a` 是 `a` 的地址
- `p` 里存的是 `a` 的地址
- `*p` 表示“去这个地址里取值”，结果是 `10`

## 2. `&` 和 `*` 的含义（必须分清）

### `&`：取地址

```c
int a = 10;
int *p = &a;
```

- `&a`：拿到 `a` 的地址

### `*`：有两种语境

1. 声明语境（类型）

```c
int *p;
```

表示 `p` 是“指向 int 的指针”。

2. 表达式语境（解引用）

```c
*p = 20;
```

表示“通过地址修改该地址中的值”。

## 3. 为什么函数里要用指针

因为 C 默认“值传递”。

```c
void f(int x) { x = 100; }   // 改不到外部变量
```

要想改外部变量，必须传地址：

```c
void f(int *x) { *x = 100; } // 能改到外部变量
```

这就是 `swap(int *a, int *b)` 的核心原因。

## 4. `swap(int *a, int *b)` 的本质

`swap` 收到的是两个变量的地址。

```c
int x = 3, y = 5;
swap(&x, &y);
```

函数内部通过 `*a`、`*b` 访问并交换的就是 `x`、`y` 本体。

## 5. 指针和数组关系（高频）

```c
int arr[5] = {1,2,3,4,5};
int *p = arr;   // 等价于 &arr[0]
```

- `arr[i]` 等价于 `*(arr + i)`
- `p[i]` 等价于 `*(p + i)`

但是：函数参数里的 `int arr[]` 会退化为 `int *arr`，所以长度要额外传入。

## 6. 指针类型为什么重要

指针类型决定两件事：
1. 解引用时按什么类型读写
2. `p + 1` 前进多少字节

例如：
- `int *p`：`p+1` 前进 `sizeof(int)` 字节
- `char *p`：`p+1` 前进 1 字节

## 7. 初学者最容易踩的坑

1. 野指针
- `int *p; *p = 10;`（错误，`p` 未初始化）

2. 空指针解引用
- `int *p = NULL; *p = 10;`（错误）

3. `scanf` 忘记传地址
- `scanf("%d", x);`（错误）
- `scanf("%d", &x);`（正确）

4. 指针和字符字面量混淆
- `'A'` 是字符，`"A"` 是字符串

## 8. `const` 和指针（先记最常用）

```c
const int *p;  // 不能通过 p 改值（值只读）
int *const p2 = &x; // p2 这个指针变量本身不能改指向
```

你当前阶段先重点掌握第一种：`const int *p`。

## 9. 调试指针的最小方法

1. 打印地址和值

```c
printf("a=%d, &a=%p, p=%p, *p=%d\n", a, (void*)&a, (void*)p, *p);
```

2. 用 `gdb` 看变量
- `break main`
- `run`
- `print a`
- `print &a`
- `print p`
- `print *p`
